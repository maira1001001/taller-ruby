#The Ruby Style Guide

##Nombres

* Usa `snake_case` para los símbolos, métodos y variables.

    ```Ruby
    # mal
    :'some symbol'
    :SomeSymbol
    :someSymbol

    someVar = 5

    def someMethod
      ...
    end

    def SomeMethod
     ...
    end

    # bien
    :some_symbol

    def some_method
      ...
    end
    ```
